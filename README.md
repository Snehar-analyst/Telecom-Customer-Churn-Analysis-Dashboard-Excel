# Telecom Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn in the telecom industry using Microsoft Excel. The objective is to identify customer segments with high churn rates and provide insights that can help improve customer retention.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Dataset
Telecom Customer Churn Dataset containing:
- Customer ID
- Telecom Partner
- Gender
- Age
- State
- City
- Estimated Salary
- Calls Made
- SMS Sent
- Data Used
- Churn Status

## Key Performance Indicators (KPIs)
- Total Customers
- Churned Customers
- Retained Customers
- Churn Rate (%)

## Dashboard Features
- Churn Analysis by Telecom Partner
- Churn Analysis by Gender
- Churn Analysis by Age Group
- Churn Analysis by State
- Salary vs Churn Analysis
- Data Usage vs Churn Analysis
- Interactive Slicers for Filtering

## Data Cleaning
- Removed duplicate records
- Handled missing values
- Corrected invalid data entries
- Standardized data formats

## Key Insights
- Identified telecom partners with the highest churn rates.
- Analyzed customer demographics associated with churn.
- Evaluated the impact of salary and data usage on customer retention.
- Highlighted high-risk customer segments for targeted retention strategies.
